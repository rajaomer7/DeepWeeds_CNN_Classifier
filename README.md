# DeepWeeds Weed Classification CNN

A Convolutional Neural Network (CNN) project for classifying in-field weed species using the **DeepWeeds dataset**. This project demonstrates data preprocessing, augmentation, transfer learning using EfficientNetB0, model training, evaluation, and Grad-CAM visualizations.
